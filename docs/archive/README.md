# Archivo histórico

Este directorio contiene documentación y pipelines legacy retirados de la raíz para mantener el proyecto operativo limpio.

## Estructura

- `legacy-azure/`: guías y notas del flujo antiguo con Azure Functions/Azure deployment.
- `legacy-devops/`: documentación antigua de Azure DevOps y configuración previa.
- `legacy-workflows/`: pipelines YAML legacy y workflows de GitHub deshabilitados.

Estos archivos se conservan por trazabilidad del TFM, pero no forman parte del flujo activo actual (Flask + Docker + Mongo + GitHub Actions).
